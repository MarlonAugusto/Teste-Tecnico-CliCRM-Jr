## Rodando projeto na maquina

```shell
git clone https://github.com/MarlonAugusto/Teste-Tecnico-CliCRM-Jr.git
```
---
#### Para baixar as depêndencias

Usando npm
```shell
  npm install
```

Usando yarn
```shell
  yarn install
```
---
#### Executar o projeto
Usando npm

```shell
  npm run dev
```

Usando yarn
```shell
  yarn dev
```

## Buildando o projeto
```shell
npm run build
firebase login
firebase init
firebase deploy
```
